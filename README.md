# 📊 Market Trend Analysis Dashboard

## 🚀 Overview

This project is an interactive data visualization dashboard built using Streamlit. It helps analyze e-commerce sales data and generate business insights.

## ✨ Features

* Interactive filters (Region, Category, Date)
* Sales trend visualization
* Profit vs Sales analysis
* Top products identification
* Smart business insights
* Download filtered data

## 🛠 Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

## 📊 Key Insights

* Identifies top-performing regions and categories
* Detects sales trends over time
* Helps in data-driven decision making

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
